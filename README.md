🌌 Gesture Arena: Rock Paper Scissors
A futuristic AI-powered gesture-controlled arcade game built using MediaPipe Hands, Three.js, GSAP, and Vanilla JavaScript.
Control the battle using real-time hand gestures through your webcam in a cinematic cyberpunk galaxy arena filled with holographic effects, neon animations, particles, meteors, and immersive arcade visuals.

✨ Features


🖐️ Real-time gesture recognition using webcam


✊ Rock gesture detection


✋ Paper gesture detection


✌️ Scissors gesture detection


🤖 AI opponent system


🌌 Futuristic cyberpunk galaxy interface


☄️ Animated stars, meteors, nebula effects


🎵 Dynamic futuristic audio engine


⚡ Combo streaks & battle effects


🎬 Cinematic transitions and countdowns


💥 Particle explosions & holographic animations


📱 Responsive design for desktop and mobile


🧠 MediaPipe hand landmark tracking


🚀 Smooth Three.js powered background engine
